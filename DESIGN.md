# 灵珠手作 · 设计思考

## 产品灵魂意象
**意象锚点**：深夜古玩铺里一盏暖灯照耀下的珠串工作台——木纹桌面上散落着玛瑙、水晶、翡翠等各色珠子，灯光透过半透明珠体折射出温润光晕，空气中浮着细微金粉。

## 气质与意象
- **关键词**：神秘、温润、奢华、匠心、玄幻
- **场景叙述**：像一位匠人在深色绸缎布景前展示手串，每颗珠子都经过精心打磨，在暖金色聚光灯下呈现真实质感。背景是深邃的夜空，偶尔有流星划过。

## 视觉策略
- **摄影方向**：珠宝产品摄影——微距镜头下的珠子质感，柔光箱打光，暗背景衬托珠体光泽
- **图形语言**：装饰性金线勾勒的几何纹样，仿古卷轴边框，水墨渐变的运势签

## 配色方案
- **背景深色**：`#0D0D1A` → `#1A1030` 渐变（夜幕蓝紫）
- **主文字**：`#F5E6C8`（暖白金色）
- **强调色**：`#D4A84B`（古铜金）
- **辅助色**：`#8B7D6B`（暗金色），`#C0A060`（浅金色）
- **点缀色**：`#FFD700`（璀璨金），`#E8C870`（暖金色高光）
- **警告色**：`#FF6B6B`（赤红—用于额度不足提示）

## 字体排版
- **中文字体**：系统默认（思源黑体/苹方），标题用粗体
- **英文数字**：系统默认
- **排版节奏**：大标题 24-28px，副标题 18-20px，正文 14-16px，标注 12px

## 动效与交互
- **运动性格**：优雅缓慢——像珠子在绸缎上缓缓滚动
- **缓动曲线**：cubic-bezier(0.25, 0.1, 0.25, 1) 或 ease-in-out
- **珠串旋转**：持续 8-10 秒完成一周的匀速旋转
- **页面转场**：淡入 + 微上滑（0.3s）
- **选中反馈**：金框放大 + 微光晕动画

## 页面结构
1. **首页**：3D 手串自动旋转展示 + 三个入口按钮（开始编串 / 签到 / 收藏）
2. **选材质**：6 种材质卡片网格 + 材质预览圆球
3. **选颜色**：6-8 色块横排 + 材质球实时更新
4. **选配珠**：配珠横向滑块 + 拖拽调整位置
5. **3D 预览**：全屏 Canvas 渲染 + 底部操作栏（确认 / 返回）
6. **结果卡**：签文 + 手串图 + 金粉装饰 + 保存/分享按钮
7. **签到页**：月历 + 连续签到标记 + 限定配色展示

## 设计禁忌
- ❌ 不要使用扁平纯色按钮（必须带微光纹理）
- ❌ 不要使用亮白色背景（全部深色）
- ❌ 不要使用圆角矩形容器以外的形状（保持统一）
- ❌ 不要让界面显得廉价或儿童化（保持神秘高端感）
- ❌ 不要使用纯黑 `#000`（用深蓝黑 `#0D0D1A`）
- ❌ 不要使用系统默认的灰白 Toast
